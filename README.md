### 基本功能
**1.显示推荐词栏top：绑定推荐词top数据，使用RecyclerView显示；**<br><br>
**2.显示历史记录：绑定历史记录数据，使用RecyclerView显示；**<br><br>
**3.显示推荐词栏bottom：绑定推荐词bottom数据，使用RecyclerView显示；**<br><br>
**4.删除历史纪录；**<br><br>
**5.隐藏和现实推荐词栏；**<br><br>
